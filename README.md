# 🎓 Smart Attendance Tracker

A modern Android application built using **Kotlin** and **Jetpack Compose** that helps students manage their attendance, predict attendance, calculate safe bunks, and interact with a built-in smart chatbot.

---

## ✨ Features

- 📅 Select semester start date
- 📚 Add and manage subjects
- 📊 Subject-wise attendance tracking
- 🗓️ Manual weekly timetable setup
- ✅ Select active class days
- 📈 Real-time attendance calculation
- 🎯 Safe bunk calculator
- 📉 Skip class prediction
- 🤖 Smart rule-based chatbot
- 📱 Floating draggable AI assistant
- 🔄 Reset all app data
- 🎨 Modern Material 3 UI

## ⚙️ How It Works

1. Select the semester start date.
2. Add subjects and their current attendance percentages.
3. Create your weekly timetable.
4. Select the days on which classes are held.
5. The app calculates:
   - Overall attendance
   - Subject-wise attendance
   - Safe bunks
   - Classes required to reach 75%
6. Use the Prediction screen to estimate attendance after skipping classes.
7. Ask the Smart Chatbot questions about your attendance and timetable.
